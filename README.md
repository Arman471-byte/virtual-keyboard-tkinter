# Virtual Keyboard (Python Tkinter)

A GUI-based virtual keyboard built using Python and Tkinter.

## Features
- Caps Lock
- Shift (temporary modifier)
- Dynamic layout rebuilding
- File saving
- OOP architecture

## How to Run
1. Make sure Python 3.10 is installed.
2. Clone the repository:
   git clone https://github.com/Arman471-byte/virtual-keyboard-tkinter.git
3. Navigate into the folder:
   cd virtual-keyboard-tkinter
4. Run:
   python Virtual-keyboard.py
